Envíos a todo México con las mejores tarifas de envío. Con este plugin, podrás cotizar cada envío, imprimir tu guía y programar la recolección.

== Description ==

Con Pakke no tendrás que preocuparte por el seguimiento de tus envíos ya que lo hacemos en todo momento y sin costo alguno.

El plugin de Pakke le ofrece a tu tienda 2 modalidades de funcionalidad, el primero le mostrara a tus clientes el costo del envío de acuerdo al tiempo de entrega: estándar, express y mismo día, permitiéndote agregar un margen de utilidad el cual te permite evitar pérdidas en tus envíos. El segundo poner un precio fijo en todos tus envíos asegurándote siempre de enviar al costo más bajo.
Además, el plugin te permitirá cotizar para elegir la mejor opción entre todas las paqueterías disponibles o te ayudará a elegir la mejor opción.

Pakke es una plataforma inteligente con la que podrás cotizar con las mejores paqueterías a nivel nacional, contarás con la cobertura más amplia, el servicio más rápido y el precio más económico del mercado.

== Installation ==

1. Copia los archivos del plugin al directorio o a través de la sección de Plugins de Wordpress.
2. Activa el plugin desde la sección de plugins de Wordpress.
3. En el dashboard de amdinistrador, accede a Woocommerce -> ajustes -> envíos.
5. Selecciona la pestaña Pakke.
6. Es necesario contar con tu token de acceso el cual deberás obtenerlo desde tu cuenta en [Pakke](https://pakke.mx/)
7. Ingresa la dirección desde donde realizarás los envíos, esto te permitirá realizar correctamente las cotizaciones.
8. Agrega Pakke en tus zonas de envío.


== Frequently Asked Questions ==

= ¿Cómo cotizo con Pakke? =

Sigue los pasos necesarios para crear tu cuenta y dar de alta tus datos, una vez que hayas completado todo tu perfil, tu tienda estará lista y podrás cotizar con Pakke. Si completas todo tu perfil podrás obtener mayores beneficios.

= ¿Cómo sacar el API Key? =

Para obtener el API Key y conectar tu tienda, ingresa a Pakke, ve a tu perfil y ahí encontrarás la sección de API Key para poderlo pegar en Woocommerce.

= ¿Por qué tengo que registrar una dirección? =

La dirección es el lugar donde van a recolectar los pedidos realizados dentro de tu tienda.

= ¿Qué es aplicar una variación de precio? =

La variación de precio te permite poner un porcentaje extra al costo de la cotización en cada uno de los servicios devueltos por Pakke, esto te permitirá tener un margen extra para realizar el envió a tu cliente sin poner en riesgo la utilidad de tu venta. Esta variación de precio no es requerida y es completamente ajustable.

Si tiene más preguntas, comuníquese con Pakke desde su sitio web.

== Funcionalidades ==

* Realiza los envíos de tus pedidos con FEDEX, UPS, Redpack, etc.
* Descarga las guías generadas en pdf desde Wordpress.
* Permite a tus clientes calcular automáticamente el costo de los envíos antes de pagar.
* Selecciona cuáles paqueterías se van a considerar en el cálculo de tus envíos.
* Modifica las etiquetas de los tipos de envío (Mismo día, Servicio express, Día siguiente, etc).
* Elige entre ofrecer a tus clientes envío con costo fijo o costo dinámico.
* Genera multi guías para los pedidos.

= Registro en Pakke =

Registrarse en Pakke es muy facil, sigue los siguientes pasos:

1. Ingresar al [sitio web de Pakke](https://seller.pakke.mx/registro)
2. Llena tus datos para crear tu cuenta
3. Inicia Sesión: Inicia sesión en tu cuenta Pakke
4. Ingresar a tu perfil para obtener tu API KEY
5. Copia el API key completo *Si no tienes uno genéralo*  

**Importante** Solo debes generarlo una vez, ya que de lo contrario tu plugin puede dejar de funcionar, si realizas alguna actualización de tu API tendrás que hacer lo mismo en tu administrador de WooCommerce.

6. Pega el API key: Regresa a Woocommerce, pega el API key y guárdalo
7. Ingresa los datos de origen de envío

== Notas ==
Asegúrate de tener completo tu perfil y validar tu número telefónico.
Es importante realizar los abonos correspondientes para que tu cuenta siempre se mantenga con saldo y así no tener ningún problema con la generación de las guías.
